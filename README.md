# HR Analytics Dashboard

## Project Overview
This project focuses on analyzing HR employee data to understand attrition,
workforce distribution.

## Tools Used
- Excel / Power BI
- SQL (for analysis)
- CSV dataset

## Key KPIs
- Employee Attrition Rate
- Department-wise Employee Count
- Average Tenure

## Key Insights
- Identified departments with high attrition
- Observed trends in employee tenure and satisfaction
- Helped HR teams understand workforce patterns

## Objective
To help HR teams make data-driven decisions for employee retention
and workforce planning.
